## Technical Architecture

### Frontend Stack
```
Framework:    React Native + Expo SDK 54
Language:     TypeScript (strict mode)
Navigation:   Expo Router (file-based)
State:        React Context API
Styling:      StyleSheet API + Theme System
UI Library:   Custom components + Ionicons
Animations:   react-native-reanimated
Testing:      Jest + Detox (E2E)
```

### Backend Integration
```
API:          RESTful (baotienweb.cloud)
Auth:         JWT tokens + Secure Store
Payments:     MoMo, VNPay, Stripe
Real-time:    WebSocket (planned)
Storage:      Expo Secure Store
Caching:      In-memory + AsyncStorage
```

### Performance Optimizations
```
✅ Lazy loading for heavy components
✅ Image optimization and caching
✅ FlatList optimization (removeClippedSubviews)
✅ Memoization with React.memo/useMemo
✅ Debouncing search inputs
✅ Throttling scroll handlers
✅ Bundle size optimization (<50MB)
✅ Code splitting with React.lazy
```

### Security Features
```
✅ Secure token storage (Expo Secure Store)
✅ HTTPS-only API calls
✅ Input sanitization (XSS prevention)
✅ Password strength validation
✅ Rate limiting
✅ Hash validation for data integrity
✅ OWASP Mobile Top 10 compliance
```
